# pdi-common-impl

このモジュールは、 PDI 実装が利用できる共通実装を提供します。
**ゲーム開発者(Akashic Engineの利用者)がこのモジュールを直接利用する必要はありません**。

## インストール

Node.jsが必要です。次のコマンドでインストールできます。

```
npm install @akashic/pdi-common-impl
```

## ビルド方法

TypeScriptで書かれています。インストール後にビルドしてください。

```sh
npm install
npm run build
```

## 利用方法

`require()` してください。

```javascript
require("@akashic/pdi-common-impl");
```

## テスト方法

```
npm test
```
